IT Full Stack Engineer

PT NUSANTARA INDONESIA

Jawa Barat, Indonesia

Deskripsi Pekerjaan

  • Mengembangkan perangkat lunak secara end-to-end, mulai dari frontend hingga backend.
  • Merancang, mengimplementasikan, dan memelihara fitur-fitur baru dalam aplikasi.
  • Bekerja sama dengan tim pengembangan untuk merancang arsitektur aplikasi yang efisien dan scalable.
  • Melakukan integrasi antara sistem backend dengan antarmuka pengguna.
  • Melakukan pengujian dan debugging untuk memastikan kestabilan dan kualitas aplikasi.
  • Mengoptimalkan kinerja aplikasi dan melakukan pemeliharaan rutin.

Kualifikasi Minimum

  • Gelar sarjana dalam Ilmu Komputer, Teknik Informatika, atau bidang terkait.
  • Pengalaman kerja minimal 2 tahun dalam pengembangan perangkat lunak, khususnya sebagai Full Stack Engineer.
  • Pemahaman yang baik tentang bahasa pemrograman seperti JavaScript, Python, atau Java.
  • Kemampuan dalam pengembangan frontend menggunakan HTML, CSS, dan JavaScript frameworks (misalnya React.js, Angular, Vue.js).
  • Pengalaman dalam pengembangan backend menggunakan teknologi seperti Node.js, Django, atau Spring Boot.
  • Kemampuan untuk bekerja dalam tim dan berkomunikasi dengan baik.
  • Kemampuan analitis yang baik dan orientasi pada solusi.

Keterampilan yang Dibutuhkan

  • Full Stack Development
  • JavaScript (Node.js)
  • HTML, CSS
  • Frontend Frameworks (React.js, Angular, Vue.js)
  • Backend Frameworks (Django, Spring Boot)
  • Database Management (SQL, NoSQL)
  • Debugging
  • Komunikasi
  • Pemecahan Masalah
