Pengembang Web

PT. CENTRAL MULIA

Denpasar, Indonesia

Deskripsi Pekerjaan

  • Mengembangkan aplikasi web front-end dan back-end menggunakan ReactJs dan NodeJs
  • Memelihara dan meningkatkan aplikasi web yang sangat skalabel, andal, dan tahan kesalahan
  • Memastikan kinerja, kualitas, keamanan, dan responsifitas aplikasi
  • Mengatasi masalah dan debug untuk mengoptimalkan kinerja
  • Melakukan peninjauan kode secara independen dan kemudian dengan rekan kerja
  • Bekerja sama dengan desainer produk, manajer, dan tim proyek untuk bereksperimen dan membangun fitur-fitur inovatif
  • Membangun aplikasi web front-end yang berfungsi dengan baik dan mulus yang berinteraksi dengan layanan API RESTful
  • Mengembangkan animasi, transisi, dan interaksi untuk UI dan elemen visual
  • Mengembangkan semua komponen pintar dan bodoh yang diperlukan untuk aplikasi
  • Tetap terkini dengan desain terbaru, tren UI, dan teknologi front-end yang muncul
  • Menerapkan pengujian unit untuk semua model, reducer, efek, dan/atau layanan yang diperlukan

Kualifikasi Minimum

  • Kandidat harus memiliki setidaknya Gelar Sarjana/Pascasarjana/Diploma/Profesional dalam Teknik (Komputer/Telekomunikasi), Ilmu Komputer/Teknologi Informasi atau setara
  • Setidaknya 4 Tahun(s) pengalaman kerja di bidang terkait diperlukan untuk posisi ini
  • Mempunyai pengalaman menggunakan Node JS dan ReactJS selama lebih dari 2 tahun
  • Pemahaman yang baik tentang teknologi front-end, termasuk HTML, CSS, platform pemrosesan CSS sisi server, seperti SASS, JavaScript dengan desain responsif, Redux Saga
  • Pemahaman yang baik tentang masalah kompatibilitas lintas browser dan cara mengatasinya
  • Pemahaman yang baik tentang alat-alat kontrol versi kode, seperti Git Pemahaman yang baik tentang penanganan permintaan asinkron, pembaruan halaman sebagian, dan AJAX

Keterampilan yang dibutuhkan

  • HTML
  • Node.js
  • CSS
  • AJAX
  • React
  • GIT
  • SQL
  • MySQL
